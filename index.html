Mis Proyectos

Carrito de compras

Ver demo Código GitHub

¿De qué se trata?

Aplicación web ASP.NET Core MVC que consiste en una plataforma para una PYME de indumentaria. Permite, por un lado, que el Empleador administre productos, categorías, marcas, empleados y stock en sucursales; y por otro lado, que el Cliente se registre, navegue el catálogo, agregue productos a un carrito, finalice sus pedidos y consulte su historial de compras.

Empleador

¿Qué puedo hacer?

  • ABM de Productos, Categorías, Marcas y Sucursales.
  • Gestionar stock por sucursal.
  • Habilitar / deshabilitar productos.
  • ABM de Empleados.
  • Consultar compras del mes y su detalle.
Cliente (Usuario)

Uso del entorno de prueba

La experiencia de cliente funciona como una tienda online demostrativa. No hay pagos reales ni se solicitan datos bancarios.

¿Cómo empezar?

  1. Ingresá a la demo desde el botón Ver demo.
  2. Hacé clic en Registrarse.
  3. Completá tus datos para crear una cuenta (demo).
  4. Luego iniciá sesión para operar con el carrito.

Nota: el registro se usa solo para habilitar el flujo de compra y el historial dentro del proyecto (demo).

¿Qué puedo hacer?

  • Navegar el catálogo sin iniciar sesión.
  • Agregar productos al carrito y modificar cantidades.
  • Finalizar compra eligiendo sucursal.
  • Acceder al historial de compras.
  • Actualizar datos de contacto.

Tecnologías

  • C#
  • HTML & CSS
  • Entity Framework Core
  • ASP.NET Core MVC
  • SQL Server / SQLite
